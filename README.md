# Venue_Vendor_Website
Full Stack Venue booking platform built with React, TypeScript, Node.js, Express, GraphQL, TypeORM, and MSSQL server. Features multi-role authentication,venue management, booking workflows, document validation, analytics dashboards, and admin reporting. 

# To start 
Go to any desinated folder/root folder for e.g admin>frontend>admin-frontend and run npm install
Do -  rm -rf .next npm run dev to remove any cache 

# Project Summary 
Venue Vendor
A full-stack venue booking and management platform that connects hirers with venue vendors through a booking workflow system.

The application supports three user roles:
Hirers – browse venues, submit booking requests, manage profiles, upload documents, and track booking history.
Vendors – manage venues, perform venue CRUD operations, approve/reject bookings, block unavailable time slots, and view booking analytics.
Admin – manage venues and vendor assignments, generate business reports, and control featured venue listings through a separate GraphQL powered admin dashboard.

**- Key Features**
Role-based authentication and authorisation
Venue CRUD operations
Booking application workflow
Document upload and validation
Reputation and credibility scoring
Vendor analytics and visual reporting
Venue suitability recommendations using keywords
Time slot availability management
GraphQL admin dashboard
Microsoft SQL Server (MSSQL) integration with TypeORM
